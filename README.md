# Fibonacci Sequence

## 📌 Description

This project is a Python program that generates the Fibonacci sequence based on user input.
The user specifies how many terms to display, and the program calculates and prints the sequence accordingly.

## 🚀 Features

* Generates the Fibonacci sequence
* Accepts user input for number of terms
* Handles different input cases (1 term, 2 terms, or more)
* Uses a `while` loop for sequence generation

## 🛠️ Technologies

* Python
